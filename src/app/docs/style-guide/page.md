## Typography

### Fonts

| Role | Font | Fallback |
|------|------|----------|
| **Headings** | Canela (serif) | Didot, Georgia, serif |
| **Body** | Inter (sans-serif) | system-ui, sans-serif |

Canela creates an elegant, editorial feel for headings. Inter keeps body text clean and readable.

### Type Scale

| Style | Size | Weight |
|-------|------|--------|
| H1 | 56–64px | Bold |
| H2 | 40–48px | Bold |
| H3 | 30–36px | Semi-bold |
| H4 | 24px | Semi-bold |
| Body | 16px | Regular |
| Body Small | 14px | Regular |
| Caption / Label | 12px, uppercase, letter-spaced | Medium |

## Visual Patterns

| Pattern | Description |
|---------|-------------|
| **Decorative bar** | A short rust-colored horizontal line (40–60px wide) used to introduce sections |
| **Feature dots** | Small circles alternating rust and green beside feature names |
| **Dark sections** | `#00182a` background with white text for contrast and visual rhythm |
| **Cards** | White background, light gray border, 12px rounded corners |
| **Italic accent** | Italicized words within serif headings for emphasis |
