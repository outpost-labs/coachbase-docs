The **File Library** is a central place to store and share documents with your clients. Upload files once and share them with one or more clients, optionally linking them to specific coaching sessions.

![File Library page showing storage capacity bar and file list](/images/file-library/01-index.png)

## Storage

Your account has a 5 GB storage soft cap. The **Storage Capacity** bar at the top of the page shows your current usage as a percentage. The exact usage is also displayed in the header next to the **Share** and **Upload** buttons.

{% callout type="warning" %}
When you approach the storage limit, a warning banner appears. Consider removing unused files to free up space.
{% /callout %}

## Uploading files

1. Click **Upload** in the top-right corner
2. Drop a file into the dashed area, or click to browse your computer
3. Click **Upload File** to save

![Upload File modal with drag-and-drop area](/images/file-library/02-upload-modal.png)

Files can be up to 25 MB each. Uploaded files are stored privately and are only accessible to you and any clients you explicitly share them with.

## Managing files

The **Library Files** table lists all uploaded files with:

- **Name** -- the original filename
- **Size** -- file size in KB or MB
- **Uploaded** -- the date the file was added
- **Shares** -- how many clients have access
- **Actions** -- **Manage** to view share details, **Download** to save a copy

Click **Manage** on any file to open a detail panel showing its current shares and options to add or remove client access.

## Sharing files with clients

Click **Share** in the top-right corner to open the share panel.

![Share Files slideover with file and client selection](/images/file-library/03-share-slideover.png)

1. Select one or more files from the **Files** list (use the search field to filter)
2. Check the clients you want to share with under **Clients**
3. Optionally associate the share with a specific **Session**
4. Click **Share Files**

Shared files appear on the client's detail page under the Files tab. You can also upload and share a file directly from a client's profile.

## Frequently Asked Questions

**What file types can I upload?**
Any file type is accepted, including PDFs, images, spreadsheets, and documents. The maximum file size is 25 MB.

**What happens when I delete a file?**
The file is permanently removed and unshared from all clients who had access. This action cannot be undone.

**Can clients upload files to me?**
Not currently. Only coaches can upload files through the File Library.
