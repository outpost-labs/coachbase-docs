The command palette gives you instant access to every page and record in CoachBase without clicking through menus. Press a keyboard shortcut, type what you are looking for, and jump straight there.

![Command palette open showing page navigation options](/images/command-palette/01-palette-open.png)

## Opening the command palette

Press **Cmd + K** (Mac) or **Ctrl + K** (Windows/Linux) from any page. The palette appears as a centered dialog with a search field.

## Navigating to pages

When you first open the palette, you see a list of all pages in CoachBase -- Dashboard, Clients, Sessions, Invoices, Packages, Scheduling, Analytics, and more. Use the arrow keys to highlight a page and press **Enter** to navigate, or click directly.

![Command palette filtered to show settings-related pages](/images/command-palette/03-palette-pages.png)

Start typing to filter the list. For example, typing "sett" narrows the results to **Settings**, **Account Settings**, and **Security Settings**.

## Searching for records

Type a name, email, phone number, or invoice number to search across all your data. Results show the record type (Client, Session, Invoice) so you can tell them apart at a glance.

![Command palette searching for a client by name, showing matching clients and sessions](/images/command-palette/02-palette-search.png)

The search checks:

- **Clients** -- Name, email, phone, first name, last name
- **Sessions** -- Client name, date
- **Invoices** -- Invoice number, client name
- **Meeting types** -- Title, location type

Select a result to navigate directly to that record.

## Search groups

For more targeted searching, scroll down past the page links to find search groups:

- **Search Clients** -- Browse and search your full client list
- **Search Sessions** -- Find sessions by client name
- **Search Invoices** -- Look up invoices by number or client
- **Search Meeting Types** -- Find meeting type configurations

Navigate into a group with **Enter** or arrow keys, then type to search within that category. Press **Backspace** on an empty search field to go back to the main list.

## Quick actions

The palette also includes quick actions for common tasks:

- **New Client** -- Jump straight to creating a new client
- **New Session** -- Schedule a new session
- **New Invoice** -- Create a new invoice
- **New Package** -- Set up a new package

## Keyboard shortcuts

| Shortcut | Action |
|---|---|
| **Cmd/Ctrl + K** | Open the command palette |
| **Arrow keys** | Navigate between results |
| **Enter** | Select the highlighted result |
| **Backspace** (empty input) | Go back to the previous level |
| **Esc** | Close the palette |

## Frequently asked questions

### Can I open the command palette from any page?

Yes. The keyboard shortcut works from any page in CoachBase as long as you are logged in.

### How many search results are shown?

The palette shows up to 5 results per type when searching across all records (clients, sessions, invoices, and meeting types). When searching within a specific group, up to 15 results are shown.

### Does the search update as I type?

Yes. Results update in real time as you type, with a short delay to avoid excessive requests.
