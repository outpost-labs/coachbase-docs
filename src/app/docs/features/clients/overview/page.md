The **Clients** page is your central hub for managing every coaching relationship. From here you can see all your clients at a glance, filter by pipeline stage, and quickly update a client's status.

![Client list showing all clients with stage, balance, and activity columns](/images/clients/01-client-list.png)

## Browsing your client list

The client table displays each client's name, email, current stage, outstanding balance, next scheduled session, last activity date, and activity score. Click any row to open that client's detail page.

Use the search bar at the top left to find clients by name or email. Results update automatically as you type.

## Filtering clients

CoachBase provides several filter controls above the table:

- **Stage** -- Filter by pipeline group (Pipeline, Engaged, Lost) or by individual stages like Lead, Contacted, or Client. Select **Archived** to view archived clients.
- **Session** -- Show clients with upcoming sessions, clients with no upcoming sessions, or all.
- **Overdue invoice** -- Toggle to show only clients who have at least one overdue invoice.

![Client list filtered to show only pipeline-stage clients](/images/clients/02-client-list-pipeline-filter.png)

When a filter is active, the filter pill appears highlighted. To clear a filter, select the default option (e.g., **Stage: All** or **Session: Any**).

### Viewing archived clients

Select **Stage: Archived** to see clients you have previously archived. Archived clients are hidden from the default view but their data is preserved.

![Client list showing archived clients](/images/clients/03-client-list-archived.png)

## Sorting

Click any column header to sort by that field. Click again to reverse the sort order. Sortable columns include:

- **Client** (alphabetical by name)
- **Stage**
- **Balance** (outstanding amount)
- **Next Session**
- **Last Activity**
- **Activity Score**

## Changing a client's stage inline

You can change a client's pipeline stage directly from the list without opening the detail page. Click the stage badge on any row to open a dropdown organized into three groups: **Pipeline**, **Engaged**, and **Lost**.

![Inline stage dropdown open on a client row](/images/clients/10-inline-stage-dropdown.png)

Select a new stage and the change saves immediately.

## Customizing columns

Click the gear icon to the right of the filter bar to access column visibility settings. You can show or hide columns like Balance, Next Session, Last Activity, and Activity Score. Your column preferences are saved automatically.

The gear menu also provides quick links to **Client Stages** and **Custom Fields** settings.

## Bulk actions

Select multiple clients using the checkboxes on the left side of each row. A bulk actions bar appears with options to:

- **Export** selected clients to CSV
- **Archive** selected clients

## Frequently Asked Questions

**How do I see all clients, including archived ones?**
The default view shows only active (non-archived) clients. Open the **Stage** filter and select **Archived** to view them.

**What does the Activity Score column show?**
Activity Score is a 0--100 metric reflecting how engaged a client is, based on session attendance (40 points), session recency (25 points), goals achieved (20 points), and recent notes (15 points). Scores are color-coded: High (80+), Good (60--79), Fair (40--59), and Low (below 40).

**Can I import clients from a spreadsheet?**
Yes. Click **Import CSV** in the top right to upload a CSV file with client data.
