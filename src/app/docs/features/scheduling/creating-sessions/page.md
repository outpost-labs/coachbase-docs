There are two ways sessions get created in CoachBase: you create them manually, or clients book them through your public booking page.

## Creating a Session Manually

1. Go to the **Sessions** page.
2. Click the **+ New Session** button in the top right.
3. Fill in the session details:
   - **Client** — Select an existing client from the dropdown.
   - **Title** — Enter a name for the session (e.g., "Weekly Check-in").
   - **Date and time** — Pick the session date and start time.
   - **Duration** — Set the session length.
   - **Status** — Choose from **Scheduled**, **Completed**, **Cancelled**, or **No Show**.
   - **Location type** — Select **In Person**, **Zoom**, **Google Meet**, **Phone**, **Custom Link**, or **Video Call**.
   - **Location details** — Enter the address, meeting link, or phone number as appropriate.
   - **Send confirmation email** — Toggle on to notify the client about the new session.
4. Click **Save** to create the session.

![New session form](/images/scheduling/10-new-session-form.png)

{% callout type="note" %}
When you create a session manually with the confirmation email toggle enabled, your client receives the same confirmation email they would get when booking through your public page — including a cancellation link.
{% /callout %}

## Viewing Session Details

Click any session from the Sessions list to open its detail page.

![Session detail page](/images/scheduling/11-session-detail.png)

The detail page displays:

- **Session title** at the top of the page
- **Client name** — Linked to the client's profile for quick access
- **Date and time** of the session
- **Duration**
- **Status badge** — Color-coded indicator (Scheduled, Completed, Cancelled, No Show)
- **Location type** and details

Two action icons appear in the top right:

- **Edit** (pencil icon) — Open the session for editing.
- **Cancel** (X icon) — Cancel the session.

## Editing a Session

1. Open the session detail page.
2. Click the **edit** (pencil) icon in the top right.
3. Update any fields — title, date/time, duration, status, location, or notes.
4. Click **Save** to apply changes.

## Changing Session Status

1. Click the **edit** icon on the session detail page.
2. Open the **Status** dropdown.
3. Select the new status: **Scheduled**, **Completed**, **Cancelled**, or **No Show**.
4. Click **Save**.

## Cancelling a Session

1. Click the **cancel** (X) icon on the session detail page.
2. Confirm the cancellation in the prompt that appears.

{% callout type="warning" %}
Cancelling a session cannot be undone. If you need to reschedule, edit the session's date and time instead.
{% /callout %}

## Frequently Asked Questions

### Can I create a session for a new client who isn't in CoachBase yet?
No. You need to add the client to CoachBase first, then create the session. Go to **Clients** and add the new client before creating the session.

### Does editing a session notify the client?
Editing a session does not automatically send a notification. If you change the date, time, or location, reach out to your client directly to let them know.

### Can I bulk-create recurring sessions?
Not currently. Each session must be created individually. For recurring clients, consider setting up a meeting type and sharing the booking link so clients can schedule their own sessions.
