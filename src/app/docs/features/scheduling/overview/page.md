The **Sessions** page is your central hub for tracking every coaching session — past, present, and future.

![Sessions list showing upcoming sessions](/images/scheduling/01-sessions-list.png)

## Navigating the Sessions Page

The page header displays "Sessions" with a **+ New Session** button in the top right. Below the header you'll find tools to quickly locate specific sessions:

- **Search** — Type a client name or session title to filter the list instantly.
- **Status** — Filter by **All**, **Scheduled**, **Completed**, **Cancelled**, or **No Show**.
- **Location** — Filter by **All**, **Video Call**, **Phone**, **In Person**, or other location types.
- **Time** — Toggle between **Upcoming** and **Past** sessions.

## Reading Session Cards

Each session appears as a card with the following details:

- **Colored avatar** — Displays the client's initial in a color-coded circle.
- **Session title** — The name of the session (e.g., "Discovery Call" or "Weekly Check-in").
- **Client name** — The client associated with the session.
- **Date and time** — When the session is scheduled.
- **Duration** — Length of the session (e.g., 30 min, 60 min).
- **Location type** — How the session will take place (Video Call, Phone, In Person, etc.).
- **Status badge** — A colored badge showing the current status (Scheduled, Completed, Cancelled, No Show).

## Quick Actions

Hover over any session card to reveal two action icons:

- **Edit** (pencil icon) — Open the session for editing.
- **Cancel** (X icon) — Cancel the session with a confirmation prompt.

![Completed sessions view](/images/scheduling/02-sessions-list-completed.png)

## Viewing Completed Sessions

Switch the **Time** filter to **Past** or the **Status** filter to **Completed** to review your session history. Completed session cards display the same details as upcoming sessions, with a "Completed" status badge.

{% callout type="note" %}
Use the **Past** time filter combined with a date range to review sessions for a specific period — helpful for invoicing or progress reviews.
{% /callout %}

## Frequently Asked Questions

### Can I export my session history?
Not directly from the Sessions page. You can view past sessions by filtering to **Past** and use your browser's print function, or access session data through your client records.

### What does the "No Show" status mean?
**No Show** indicates a client did not attend a scheduled session. You can set this status manually by editing the session after the scheduled time has passed.

### How do I find a specific client's sessions?
Type the client's name in the search bar at the top of the Sessions page. The list filters in real time as you type.
