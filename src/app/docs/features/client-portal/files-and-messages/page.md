The portal gives clients two ways to stay connected between sessions: a file library for shared resources and a messaging thread for direct communication.

## Files

![Files page showing shared documents](/images/client-portal/06-files.png)

The Files page displays every file you've shared with the client. Each file shows the name, file size, and the date it was shared.

### Downloading files

Click the **Download** button next to any file to save it. Files are served securely -- each client can only access files explicitly shared with them.

### New file indicators

An orange dot appears next to files that were added since the client's last portal visit, making it easy to spot new resources.

### Empty state

When no files have been shared yet, clients see: "No files shared yet. Files your coach shares with you will appear here."

## Messages

![Messages page with compose area](/images/client-portal/07-messages.png)

The Messages page provides a threaded conversation between you and your client. Messages from you appear on the left with your name. Messages from the client appear on the right. Each message shows a timestamp.

### Sending messages

Clients type a message in the text area at the bottom of the page and click **Send**. Messages can be up to 5,000 characters long. There is no subject line or threading -- it's a simple, linear conversation.

### Rate limiting

To prevent spam, clients are limited to 10 messages per hour. This is generous for normal coaching communication but prevents abuse.

### Email bounce handling

If a client's email has bounced (indicating a delivery problem), the message compose area is disabled with an explanation. This prevents clients from sending messages that cannot generate email notifications to you.

### Unread message count

When the client has unread messages from you, a count badge appears on the **Messages** link in the navigation bar. This updates automatically as new messages arrive.

## Frequently Asked Questions

- **Do I get notified when a client sends a message?** Yes. Client messages generate a notification in your CoachBase inbox and can trigger email notifications depending on your notification settings.
- **Can clients send file attachments?** Not currently. Clients can download files you share with them, but file uploads from the portal are not supported.
- **Can I disable messaging for a specific client?** Yes. Use the portal visibility settings to hide the messages section for any client.
